## About Language Model Diligence

Language Model Diligence is the practice of applying established model governance disciplines to language model deployments. The Mossrake Language Model Diligence framework provides a structured approach to understanding AI process exposure, grounded in established practices from quantitative finance, pharmaceutical manufacturing, avionics safety engineering, and software supply-chain security.

The framework measures two things:
- **Consequence** (Q1) — how much is at stake if the model produces a wrong output
- **Coverage** (Q2–Q5) — the visibility, controls, and recovery mechanisms in place across four independent dimensions: Identity, Drift, Verification, and Recovery

The gap between consequence and coverage is your **exposure**. The assessment produces one of three tiers: **Calibrated** (coverage matches consequence), **Partial** (material gaps exist), or **Uncertain** (coverage cannot be determined).

For the full framework, scoring methodology, intellectual lineage, and worked examples, see the Language Model Diligence whitepaper at [mossrake.ai/language-model-diligence](https://mossrake.ai/language-model-diligence).
